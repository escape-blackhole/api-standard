### 用户登录 

> request    id可为数组  如: 01,02,02
```
url: /user/login
type: delete
```
    
> response
```
{
    "status": 0,
    "msg": "删除成功",
    "data": {},
    "success": true
}

{
    "status": 1,
    "msg": "删除失败",
    "data": null,
    "success": false
}
```
